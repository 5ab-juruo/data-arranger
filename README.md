# data-arranger
A Data Arranger for Competitive Programming Debugging

---

## What is this?
This is an easy program for arranging datas for debugging use, especially helpful for competitive programming.

You can test your program, add one more couple of data and configure your settings through only one console.

## How can I use it?
Download the `main.cpp` in the root directory and compile it. Then you can run it on your computer.

## I found a bug!
It's welcome to issue through Github. Feel free to provide suggestions!

## About more
This project is currently under development so it cannot fit many systems or environments well.

This program can be only executed on Windows stably.
